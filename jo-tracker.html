First-time setup — your workouts sync to a private GitHub Gist so your data is always safe.

1
Go to GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)
2
Give it any name, tick only the gist scope, set expiry to No expiration, click Generate token.
3
Paste the token below. A private Gist will be created automatically.
